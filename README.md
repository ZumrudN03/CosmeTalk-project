# CosmeTalk

CosmeTalk is a fullstack beauty review and blog platform built with React, Node.js, Express, and MongoDB.  
Users can explore makeup and skincare products, read blog content, check detailed review pages, and leave comments.  
An admin panel is available for managing products and blog content.

## Features

- Makeup products listing
- Skincare products listing
- Product detail and review pages
- Blog section with featured content
- User authentication
- Admin panel for product and blog management
- MongoDB database integration

## Tech Stack

### Frontend
- React
- SCSS
- Vite

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
